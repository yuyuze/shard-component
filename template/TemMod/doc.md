## 概述

`Mod` 组件 `theme` 可以自由切换主题，除特殊主题外传入参数基本都固定，特殊主题会具备特殊参数

## API

### Scroll

| 参数 | 说明     | 类型             | 默认值 | 版本  |
| ---- | -------- | ---------------- | ------ | ----- |
| val  | 值(数据) | number \| string | 0      | 0.0.3 |
