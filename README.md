<h1 align="center">
  VC
</h1>

为数据可视化项目提供的一款组件库。[文档地址](http://ys.hztujing.com:9999/demo/VC-DOC/)

## 特性

- 提炼自数据可视化的视觉风格。
- 开箱即用的高质量 Vue 组件。

## 支持环境

- 支持 Vue 3

## 安装

### 使用 npm 或 yarn 安装

```bash
$ npm install tj-vc --save
```

```bash
$ yarn add tj-vc
```

如果你的网络环境不佳，推荐使用 [cnpm](https://github.com/cnpm/cnpm)。

```
// 目录
site  // 文档项目存放地
site/scripts  // 生成路由文件
components // 组件
plugin // 解析在 components 中的 markdown 文件
```
