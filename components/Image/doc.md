## 概述

图片

## API

### Text

| 参数     | 说明             | 类型             | 默认值 | 版本 |
| -------- | ---------------- | ---------------- | ------ | ---- |
| width    | 宽               | string \| number | -      | -    |
| height   | 高               | string \| number | -      | -    |
| src      | 图片路径         | string           | -      | -    |
| fallback | 加载失败默认图片 | string           | -      | -    |
| lazy     | 启用懒加载       | boolean          | true   | -    |
| loading     | 是否开启加载动画 | boolean          | true   | -    |
| preview  | 启用预览功能     | boolean          | true   | -    |
